FAQ

よくあるご質問

JDGPの活動や参加方法、支援についてよくいただくご質問をまとめました。お探しの情報が見つからない場合は、お気軽にお問い合わせください。

団体・参加資格について

Q

どんな団体ですか？

add
A

JDGP（NPO法人日本障害者ゴルフ選手会 / Japan Disabled Golf Players Association）は、障がい者ゴルファーが安心してゴルフを楽しめる環境づくりを目的とするNPO法人です。

2012年に任意団体として発足し、2013年6月14日にNPO法人として認証されました。現在は会員数 約70名、代表者は小林 英喜です。

Q

どのような障がいがあれば参加できますか？

add
A

障がいの種類・等級を問わず、ゴルフに挑戦したい方を幅広く対象としています。身体・視覚・聴覚・知的・精神などあらゆる障がいに対応できるよう努めています。

合理的配慮や用具・練習環境のご相談も承りますので、まずはお問い合わせフォームよりご連絡ください。

会員制度について

Q

会員になるにはどうすればいいですか？

add
A

参加するページから会員登録のご案内をご確認いただけます。選手会員・ボランティア会員など、関わり方に応じた参加形態をご用意しています。

Q

ゴルフ初心者でも会員になれますか？

add
A

はい、経験や技量を問わずご参加いただけます。練習会・交流会など、初めての方でも参加しやすい機会を提供しています。

用具や練習環境のご相談も承ります。お気軽にお問い合わせください。

参加支援について

Q

会員でなくても参加支援は受けられますか？

add
A

会員外の方もご相談いただけます。ただし、支援対象・支援範囲は案件ごとの審査、予算、優先度により決定いたしますのでご了承ください。

まずは参加支援フォームよりご相談ください。

Q

海外大会への参加も支援してもらえますか？

add
A

US Adaptive Open、G4D Open などの主要国際大会への出場支援にも力を入れています。エントリー・書類準備のサポート、渡航・宿泊等の費用の一部補助、事前準備の情報提供などを行います。

支援範囲は案件ごとに個別に判断いたしますので、詳しくは参加支援フォームよりご相談ください。

寄付について

Q

寄付はどのような方法でできますか？

add
A

現在は銀行振込（ゆうちょ口座）による寄付を受け付けています。任意で寄付申込フォームにご記入いただくことも可能です。

詳しくは寄付のご案内をご覧ください。

Q

寄付者の名前は公開されますか？

add
A

個人寄付者の方のお名前は、ご希望いただいた場合のみ活動報告内でお礼として掲載いたします。寄付者一覧ページは設けておりません。

匿名での寄付も歓迎しております。

企業協賛・連携について

Q

企業協賛にはどのような種類がありますか？

add
A

以下の3種類をご用意しています：

  • 資金協賛：年額／単発
  • 物品協賛：用具・備品・景品等
  • 人的協賛：運営、撮影、制作、事務等のボランティア参加

また、パラアスリート個人スポンサー（選手紹介）もご相談いただけます。詳しくは企業協賛・連携ページをご覧ください。

Q

協賛企業のロゴはウェア等に掲載できますか？

add
A

ウェア・キャップ等へのロゴ掲載は、大会規定や契約等により可否が変わります。事前にご相談いただき、個別に調整いたします。

協賛ページへのロゴ掲載（企業公式サイトへのリンク付き）は基本対応となります。

お問い合わせについて

Q

お問い合わせへの返信はどのくらいかかりますか？

add
A

返信には1週間程度お時間をいただいております。内容によってはさらにお時間をいただく場合がありますので、あらかじめご了承ください。

お急ぎの場合は、お問い合わせフォームにその旨ご記載いただけますと幸いです。

他にもご不明な点がございますか？

上記に記載のないご質問は、お問い合わせフォームよりお気軽にお寄せください。
